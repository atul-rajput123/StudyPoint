# StudyPoint
StudyPoint: A Comprehensive Study and Exam Management Platform

Technologies Used:
Java Servlet: For handling server-side logic and user interactions.
JSP (JavaServer Pages): To design and manage dynamic user interfaces.
JDBC (Java Database Connectivity): For database operations like data retrieval and manipulation.
Apache Tomcat: Application server for deploying the project.
HTML, CSS, JavaScript: To create responsive and interactive web pages.
Features of the Platform:
1. User Management
Registration and Login System:

Users can register with their details.
Accounts are activated only after admin approval, ensuring controlled access.
Admin Approval:

Admin reviews registrations and approves or rejects accounts as necessary.
2. Course Management
Course Listings:

Provides a detailed list of available courses.
Each course includes downloadable study notes (PDFs) and video content.
Download Facility:

Students can download course materials for offline learning.
3. Examination System
Mock Tests:

Students can attempt mock tests for self-assessment and practice.
Online Exams:

Allows organizing secure online exams with configurable patterns, timers, and sections.
Result Management:

Publishes results after the exam. Results are available for viewing in the user’s dashboard.
4. Admin Panel for Data Management
User Data Management:

Admin can perform CRUD operations (Create, Read, Update, Delete) on user data.
Includes features to:
Insert new user records into the database.
Update existing user details (e.g., permissions, roles).
Delete user records when no longer needed.
Search for specific users and view their details.
Course and Exam Data Management:

Admin can add, edit, or remove courses, notes, and exam-related information.
5. Help and Support
Help Section:
Offers email, contact information, and a chatbot for instant assistance.
6. Profile Management
User Profile:
Displays user details, including name, email, and course enrollments.
7. Advanced Search
Google Custom Search JSON API:
Enables efficient searching of study materials, topics, and courses using Google-powered search functionality.
8. Security Features
Controlled Login:

Users cannot log in until their accounts are approved by the admin.
Camera Monitoring:

Exam sessions include camera access for live monitoring, ensuring secure exam environments.
